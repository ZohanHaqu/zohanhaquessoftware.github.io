AudioBang

A simple sound utility with text-to-speech and a basic music player.

  • Convert text into speech
  • Play audio files easily
  • Lightweight and efficient
  • Easy-to-use interface

AudioBang is written in Python, but it is secretly written in C++ because it's based on PyQt, which is written in both Python and C++.
